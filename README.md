## **WezTerm Shortcut Cheat Sheet**

### **Misc / Utilities**

| Key | Action |
| --- | --- |
| F1 | Activate Copy Mode |
| F2 | Activate Command Palette |
| F3 | Show Launcher |
| F4 | Show Launcher (Tabs Only) |
| F5 | Show Launcher (Workspaces Only) |
| F11 | Toggle Fullscreen |
| F12 | Show Debug Overlay |
| SUPER+Space | Clear Current Line |
| SUPER+f | Search Text |

---

### **Copy / Paste**

| Key | Action |
| --- | --- |
| Ctrl+Shift+c | Copy to Clipboard |
| Ctrl+Shift+v | Paste from Clipboard |

---

### **Tabs**

**Spawn / Close**

| Key | Action |
| --- | --- |
| Ctrl+Shift+t | Spawn Tab (DefaultDomain) |
| SUPER_REV+w | Close Current Tab |

**Navigation / Move**

| Key | Action |
| --- | --- |
| SUPER+[ | Previous Tab |
| SUPER+] | Next Tab |
| SUPER_REV+[ | Move Tab Left |
| SUPER_REV+] | Move Tab Right |

**Tab Bar / Titles**

| Key | Action |
| --- | --- |
| SUPER+9 | Toggle Tab Bar |
| SUPER+0 | Rename Tab |
| SUPER_REV+0 | Undo Rename |

---

### **Windows**

| Key | Action |
| --- | --- |
| SUPER+n | Spawn New Window |
| ALT+= | Increase Window Size |
| ALT+- | Decrease Window Size |
| SUPER_REV+Enter | Toggle Fullscreen |

---

### **Panes**

**Split**

| Key | Action |
| --- | --- |
| SUPER+\ | Split Vertical |
| SUPER_REV+\ | Split Horizontal |

**Zoom / Close**

| Key | Action |
| --- | --- |
| SUPER+Enter | Toggle Pane Zoom |
| SUPER+w | Close Current Pane |

**Navigation (Alt + H/J/K/L)**

| Key | Action |
| --- | --- |
| ALT+h | Move Left |
| ALT+l | Move Right |
| ALT+k | Move Up |
| ALT+j | Move Down |

---

### **Scrolling**

| Key | Action |
| --- | --- |
| SUPER+u | Scroll Up 5 Lines |
| SUPER+d | Scroll Down 5 Lines |
| PageUp | Scroll Up 0.75 Pages |
| PageDown | Scroll Down 0.75 Pages |

---

### **Leader Mode**

**Resize Panes (Leader + p)**

| Key | Action |
| --- | --- |
| h | Adjust Left |
| j | Adjust Down |
| k | Adjust Up |
| l | Adjust Right |
| Escape / q | Exit Key Table |

**Resize Fonts (Leader + f)**

| Key | Action |
| --- | --- |
| k | Increase Font Size |
| j | Decrease Font Size |
| r | Reset Font Size |
| Escape / q | Exit Key Table |

---

### **Mouse**

| Trigger | Action |
| --- | --- |
| Ctrl + Left Click | Open Link at Cursor |
